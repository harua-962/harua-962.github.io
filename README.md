##使用技術

このサイトは、以下の技術を使用して構築されています。

* **HTML5**
* **CSS3**
    * レスポンシブデザイン (Flexbox, Grid)
* **JavaScript (ES6+)**
    * `History`, `Skills`, `Projects` セクションのDOM操作による動的コンテンツ生成
    * スムーズスクロール、スクロールスパイの実装

##こだわった点

* **保守性:** `script.js` 内の配列データを変更するだけで `History` や `Skills` の内容が更新されるよう、HTMLのハードコーディングを避け、JavaScriptによる動的生成を採用しました。
* **UI/UX:** スムーズスクロールや、今見ているセクションをハイライトするスクロールスパイを実装し、操作性を高めました。
